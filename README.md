# Cryptography

### This repository contains practices and projects related to the subject Cryptography at ESCOM - IPN.

Some of the topics studied were:

* Modular Arithmetic.

* Affine cipher.

* Extended Euclidean algorithm.

* Hill cipher.

* Modes of operation.
    * ECB.
    * CBC.
    * CTR.
    * OFB.
    * CFB.

* Block ciphers.
    * DES
    * AES

* Hash functions.

* RSA.

* Digital Signature Algorithm.
